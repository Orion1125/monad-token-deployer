# monad-token-deployer

## how to deploy
- clone repo or download zip.
- install packages with 
`npm install`
- run `npx hardhat` and create an empty config file
- run 
`npx hardhat compile`
- create your environment variable file and create variables named
`MONAD_RPC=`
`DEPLOYER_PRIVATE_KEY=`
- run `node deploy.js`
